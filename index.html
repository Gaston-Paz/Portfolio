AppTurn

Es una aplicación web desarrollada con C# en .Net Framework y con base de datos en SQL Server.AppTurn es un soft pensado en la gestión administrativa de los turnos de una clínca, que permite manejar la agenda de los distintos profesionales. Tiene varios perfiles de acceso que otorgan distintos tipos de funcionalidades: Administrador, recepcionista y médico.
